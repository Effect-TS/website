## Technology

[**Effect v4 Beta**](https://effect.website/blog/releases/effect/40-beta/) is officially live! Here are the key highlights:

- **Rewritten runtime** with lower memory overhead, faster execution, and simpler internals. Bundle sizes have dropped dramatically â€” a minimal program combining Effect, Stream, and Schema went from ~70 kB in v3 to ~20 kB in v4.
- **Unified versioning** across all ecosystem packages, eliminating version mismatch confusion.
- **Consolidated architecture** â€” many previously separate packages (`@effect/platform`, `@effect/rpc`, `@effect/cluster`) have been merged into the core `effect` package.
- **Unstable modules** via `effect/unstable/*` import paths, enabling faster iteration on emerging APIs before they graduate to stable status.
- **v3 compatibility** â€” the core programming model remains unchanged, and Effect v3 will continue receiving maintenance alongside v4.
